# RGBA/HEX Color Converter for Panic Nova

p53*designが提供するRGBAとHEXカラーコードを相互変換するNova拡張機能です。

## 機能

- **RGBA → HEX変換**: `rgba(255, 0, 0, 0.5)` → `#ff000080`
- **HEX → RGBA変換**: `#ff0000` → `rgb(255, 0, 0)`
- **自動検出変換**: カラー形式を自動で判定して変換
- **クリップボード対応**: クリップボード内のカラーコードも変換可能

## 使い方

### エディタ内のテキストを変換
1. エディタでカラーコードを選択
2. 以下の方法で変換を実行：

### メニューから
- **Editorメニュー**:
  - RGBAをHEXに変換
  - HEXをRGBAに変換
  - カラー形式を自動検出して変換

- **Extensionsメニュー**（クリップボード用）:
  - クリップボードのRGBAをHEXに変換
  - クリップボードのHEXをRGBAに変換
  - クリップボードのカラーを自動変換

### ショートカットキー
- **⌃⌘H**: RGBAをHEXに変換
- **⌃⌘R**: HEXをRGBAに変換
- **⌃⌘C**: カラー形式を自動検出して変換
- **⌃⌘⇧H**: クリップボードのRGBAをHEXに変換
- **⌃⌘⇧R**: クリップボードのHEXをRGBAに変換
- **⌃⌘⇧C**: クリップボードのカラーを自動変換

### コマンドパレット
- **⇧⌘P** でコマンドパレットを開き、上記のコマンド名を検索

## 対応フォーマット

### RGBA形式
- `rgb(255, 0, 0)`
- `rgba(255, 0, 0, 0.5)`
- `rgba( 255 , 0 , 0 , 0.5 )` （スペースに対応）

### HEX形式
- `#ff0000`
- `#f00` （短い形式）
- `#ff000080` （アルファ値付き）
- `#f08` （短いアルファ値付き）
