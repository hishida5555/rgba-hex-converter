# Nova拡張機能テスト手順

## 🧪 テスト準備

### 1. 拡張機能のインストール
1. Novaを起動
2. メニューから `Extensions > Extension Library` を選択
3. 右下の `+` ボタンをクリック
4. `Install Extension from Folder...` を選択
5. `/Volumes/WorkSSD/Windsurf/rgba-hex-NovaExtension` フォルダを選択
6. インストール完了を確認

### 2. 拡張機能の有効化確認
- `Extensions > Extension Library` に「RGBA/HEX Color Converter」が表示されていることを確認
- ステータスが「Enabled」であることを確認

## 🎯 テストケース

### テスト1: 基本的なRGBA→HEX変換
1. 新しいファイルを作成（`⌘N`）
2. 以下のテキストを入力：`rgb(255, 0, 0)`
3. テキストを選択
4. `⌃⌘H` を押すか、Editorメニューから「RGBAをHEXに変換」を選択
5. **期待結果**: `#ff0000` に変換される

### テスト2: アルファ値付きRGBA→HEX変換
1. `rgba(255, 0, 0, 0.5)` を入力
2. 選択して `⌃⌘H` を押す
3. **期待結果**: `#ff000080` に変換される

### テスト3: スペース付きRGBA→HEX変換
1. `rgba( 128 , 64 , 192 , 0.75 )` を入力
2. 選択して `⌃⌘H` を押す
3. **期待結果**: `#8040c0bf` に変換される

### テスト4: 基本的なHEX→RGBA変換
1. `#ff0000` を入力
2. 選択して `⌃⌘R` を押す
3. **期待結果**: `rgb(255, 0, 0)` に変換される

### テスト5: 短いHEX形式変換
1. `#f00` を入力
2. 選択して `⌃⌘R` を押す
3. **期待結果**: `rgb(255, 0, 0)` に変換される

### テスト6: アルファ値付きHEX→RGBA変換
1. `#ff000080` を入力
2. 選択して `⌃⌘R` を押す
3. **期待結果**: `rgba(255, 0, 0, 0.50)` に変換される

### テスト7: 自動検出変換（RGBA→HEX）
1. `rgba(0, 255, 128, 0.3)` を入力
2. 選択して `⌃⌘C` を押す
3. **期待結果**: `#00ff804d` に変換される

### テスト8: 自動検出変換（HEX→RGBA）
1. `#00ff80` を入力
2. 選択して `⌃⌘C` を押す
3. **期待結果**: `rgb(0, 255, 128)` に変換される

### テスト9: 複数選択変換
1. 複数行に以下を入力：
   ```
   rgb(255, 0, 0)
   #00ff00
   rgba(0, 0, 255, 0.5)
   ```
2. すべてを選択（`⌘A`）
3. `⌃⌘C` を押す
4. **期待結果**: すべての行が適切に変換される

### テスト10: エラーハンドリング
1. `invalid-color` を入力
2. 選択して変換コマンドを実行
3. **期待結果**: 警告メッセージが表示される

## 🎮 実行方法の確認

### メニューからの実行
1. テキストを選択
2. `Editor` メニューを確認
3. 以下のコマンドが表示されていることを確認：
   - RGBAをHEXに変換
   - HEXをRGBAに変換
   - カラー形式を自動検出して変換

### ショートカットキーの確認
- `⌃⌘H`: RGBA→HEX変換
- `⌃⌘R`: HEX→RGBA変換
- `⌃⌘C`: 自動検出変換

### コマンドパレットからの実行
1. `⇧⌘P` でコマンドパレットを開く
2. 「RGBA」や「HEX」で検索
3. コマンドが見つかり、実行できることを確認

## 🔍 期待される動作

### 正常時
- 選択テキストが即座に変換される
- カーソル位置が保持される
- 複数選択が正しく処理される

### エラー時
- 無効なカラーコードの場合、警告メッセージが表示される
- 元のテキストは変更されない

## 🐛 トラブルシューティング

### 拡張機能が表示されない場合
1. Novaを再起動
2. `Extensions > Extension Library` で拡張機能が有効になっているか確認
3. 拡張機能を一度無効化して再度有効化

### ショートカットが効かない場合
1. 他の拡張機能と競合していないか確認
2. Novaの環境設定でキーボードショートカットを確認

### 変換が正しく動作しない場合
1. 選択テキストに余分な文字が含まれていないか確認
2. Novaのコンソール（`Help > Show Console`）でエラーを確認
